# ThiSpot 최종 기획안

## 한 줄 정의

**ThiSpot은 오늘의 컬러 미션으로 사용자를 걷게 만들고, AI가 사진 인증과 캐릭터 기반 숏폼 리워드를 생성해 걷기를 습관화하는 미션형 워킹 앱입니다.**

## 문제 정의

걷기는 가장 쉬운 운동이지만, 혼자 걷는 사람은 금방 지루해지고 지속할 이유를 잃습니다. 기존 걷기 앱은 걸음 수, 거리, 칼로리처럼 숫자만 기록하기 때문에 운동 직후의 감정적 보상이나 공유하고 싶은 결과물을 만들지 못합니다.

핵심 문제는 “걷기 기록이 부족하다”가 아니라 **걷기를 다시 하고 싶게 만드는 즉각적 미션과 보상이 부족하다**는 점입니다.

## 타깃 사용자

- 운동 앱을 깔았지만 오래 쓰지 못한 20대
- 러닝은 부담스럽지만 걷기는 가능한 운동 초보자
- 산책 중 사진 찍기와 SNS 공유를 좋아하는 사용자
- 스트레스 해소용 짧은 산책 루틴이 필요한 사용자
- 혼자 걷는 시간이 지루한 사용자

## 핵심 인사이트

사람들은 칼로리 숫자만으로 매일 걷지 않습니다.

하지만 **오늘 할 미션**, **AI 인증**, **캐릭터 성장**, **공유 가능한 결과물**이 있으면 걷기는 반복 가능한 놀이가 됩니다.

## 트렌드 근거

해외에서는 Color Walk가 TikTok, RedNote 등에서 mindful walking 트렌드로 확산되고 있습니다. 사용자는 산책 전에 색 하나를 고르고, 걷는 동안 그 색을 가진 사물을 찾고, 사진이나 영상으로 기록합니다.

- Bustle은 2025년 2월 기사에서 Color Walk를 색을 찾는 산책형 scavenger hunt로 소개했고, TikTok 관련 게시물이 1,100만 개 이상이라고 언급했습니다.
- Tom's Guide는 Color Walking을 grounding technique과 비슷한 활동으로 설명했습니다. 색 하나에 주의를 고정하면 과도한 생각에서 벗어나 현재 주변 환경에 집중하게 됩니다.
- ECNS는 2026년 봄 중국 RedNote에서 ColorWalk 관련 조회수가 2억 1천만 회 이상이라고 보도했습니다.
- BMJ 2024 메타분석은 walking/jogging이 우울 증상 완화에 유의미한 효과가 있다고 보고했습니다.
- Molecular Psychiatry 2022 연구는 자연 속 걷기가 스트레스 관련 뇌 영역인 amygdala 활성도 감소와 연결된다고 보고했습니다.

## 솔루션

ThiSpot은 걷기를 “오늘의 색을 찾는 미션”으로 바꿉니다.

사용자는 앱에 들어오면 자신의 캐릭터와 오늘의 컬러 미션을 받습니다. 걷기를 시작하면 GPS, 걸음 수, 시간이 기록되고, 사용자는 걷는 중 오늘의 색이 담긴 사진을 촬영합니다. AI는 사진이 미션에 맞는지 색상과 오브젝트를 기준으로 인증합니다. 걷기가 끝나면 캐릭터, 사진, 걷기 기록을 조합해 숏폼 영상 또는 이미지 리포트를 생성하고, 사용자는 이를 SNS에 공유할 수 있습니다.

## 핵심 사용자 플로우

```text
앱 진입 / 데모 로그인
-> 사용자 캐릭터 등장
-> 오늘의 컬러 미션 추천
-> 캐릭터 의상이 오늘의 컬러로 변경
-> 걷기 시작
-> GPS / 걸음 수 / 시간 측정
-> 걷는 중 사진 촬영
-> VisionMissionAgent가 사진 미션 적합도 판단
-> 걷기 종료
-> 거리 / 걸음 수 / 시간 / 획득 사진 수 표시
-> DiscoveryAgent가 새 스팟/공유 스팟 판단
-> RewardAgent가 배지 지급
-> ContentGenerationAgent가 숏폼 영상 또는 이미지 리포트 생성
-> SNS 공유
```

## 핵심 AI 경험

ThiSpot의 AI는 단순히 사진을 꾸미는 장식이 아닙니다.

AI는 사용자의 실제 오프라인 행동을 검증하고, 그 행동을 보상 콘텐츠로 변환합니다.

- `PersonalWalkAgent`: 오늘의 컬러 미션과 캐릭터 컬러 상태 결정
- `VisionMissionAgent`: 사진 속 색상과 오브젝트를 분석해 미션 적합도 인증
- `DiscoveryAgent`: GPS와 비전 결과를 기반으로 새 스팟/공유 스팟 판단
- `RewardAgent`: 배지와 결과 카피 생성
- `ContentGenerationAgent`: 캐릭터, 사진, 걷기 기록으로 숏폼 영상/이미지 리포트 생성

- 캐릭터 의상/테마 색 변경

- 숏폼 영상 preview 또는 이미지 리포트 생성

후순위:

## 리텐션 요소

- `Daily Color Quest`: 매일 다른 색 미션
- `Character Growth`: 미션 수행에 따라 캐릭터 의상/배지 성장
- `Mission Proof`: AI가 실제 사진 인증
- `AI Memory Report`: 걷기 종료 후 공유 가능한 결과물
- `First Finder`: 새로운 컬러 스팟 첫 발견 보상
- `Weekly Color Recap`: 한 주 동안의 컬러 기록 요약

## 기술 차별점

1. **Vision-Grounded Mission Proof**
   사용자가 실제로 오늘의 색을 발견했는지 AI가 사진 기반으로 인증합니다.

2. **Offline Action to Shareable Reward**
   걷기, 사진, 캐릭터, 위치 데이터를 숏폼 영상/이미지 리포트로 변환합니다.

3. **Geo-Color Discovery**
   사진 속 색상 분석과 위치 데이터를 결합해 새로운 컬러 스팟을 발견합니다.

## 사업성

ThiSpot은 개인 워킹 앱으로 시작하지만, 장기적으로는 **브랜드와 지역이 사람들을 오프라인 공간으로 움직이게 만드는 컬러 미션 플랫폼**이 될 수 있습니다.

수익 모델:

- 브랜드 컬러 챌린지: Nike Red Walk, Starbucks Green Walk, Toss Blue Walk
- 지역/관광 컬러 미션: 성수 Blue Route, 제주 Green Walk, 캠퍼스 Color Hunt
- 캐릭터 스킨, 리포트 템플릿, 시즌 배지
- 기업/학교/지자체용 단체 챌린지 운영

GTM:

- 대학 캠퍼스 컬러 챌린지
- 회사 점심 산책 챌린지
- 워킹/러닝 크루 미션
- 지역 축제 컬러 스탬프 투어
- 브랜드 SNS UGC 캠페인

## 발표용 30초 문제 설명

기존 걷기 앱은 내가 얼마나 걸었는지만 기록합니다. 하지만 사람들이 매일 걷지 못하는 이유는 숫자를 몰라서가 아니라, 혼자 걷는 시간이 지루하고 끝난 직후 공유할 만한 보상이 없기 때문입니다. ThiSpot은 오늘의 컬러 미션과 캐릭터를 통해 사용자를 걷게 만들고, AI가 실제 사진을 인증해 걷기를 공유 가능한 리워드로 바꿉니다.

## 라이브 데모 90초

```text
1. ThiSpot을 열면 캐릭터가 등장합니다.
2. 오늘의 컬러 미션 Blue가 추천됩니다.
3. 캐릭터 의상이 Blue로 바뀝니다.
4. 걷기 시작 버튼을 누르면 GPS, 걸음 수, 시간이 측정됩니다.
5. 걷는 중 파란 하늘 사진을 촬영합니다.
6. VisionMissionAgent가 Blue sky, 87% match라고 인증합니다.
7. 걷기를 종료하면 1.24km, 1,843 steps, 12m 결과가 표시됩니다.
8. DiscoveryAgent가 New Blue Spot을 발견했다고 알려줍니다.
9. RewardAgent가 Blue Finder 배지를 지급합니다.
10. ContentGenerationAgent가 캐릭터와 사진을 활용한 Blue Walk 리포트를 생성합니다.
```

## 차별점 60초

ThiSpot은 단순 걷기 기록 앱이 아닙니다. 첫째, AI가 사용자의 오프라인 행동을 사진으로 인증합니다. 둘째, 인증된 걷기 기록을 캐릭터 기반 숏폼 리워드로 변환해 즉각적인 보상을 제공합니다. 셋째, 이 구조는 개인의 걷기 습관화뿐 아니라 브랜드, 지역, 캠퍼스가 사람들을 오프라인 공간으로 움직이게 만드는 컬러 미션 플랫폼으로 확장될 수 있습니다.

## 최종 피치

**ThiSpot은 걷기를 숫자 기록에서, AI가 인증하고 캐릭터가 보상하는 컬러 미션 워킹 경험으로 바꿉니다.**
